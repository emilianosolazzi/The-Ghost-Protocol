# The-Ghost-Protocol
GhostProtocol is a dark-fun, on-chain GameFi memecoin experiment where users submit evidence, earn GHOSTED rewards based on severity, and unlock story content through token-driven progression. Built with Solidity, React, and TypeScript, it blends verifiable on-chain actions with social gameplay and a bold cyber-noir interface.
